Nesta documentação iremos explicar como criar uma APPI e uma FPO para o funcionamento da tela de agendamento de vagas reguladas.

  • APPI
  • FPO

Passos para criar uma APPI:

Para criar uma APPI vá em Configuração > Configuração da CMC > Configurar Limites Físicos/Financeiro. Lá selecione a última competência existente e coloque no campo competência inicial a próxima competência. Exemplo: No combo de PPI's já existentes você vai selecionar a ultimá competência, no nosso caso é 09/2013 até 08/2014 e no campo de competência inicial logo a baixo você vai colocar 09/2014, depois é só next, next e finish. Após estes passos já deve aparecer a nova PPI.

OBS: Para criar um APPI do mês 09 devemos estar no mês 08 ou antes, caso já tenha passado este mês, é preciso trocar a data da máquina para o mês anterior ao que você vai criar.
Agora é preciso definir capacidade programada. Vá até Configuração > Configuração da CMC > Definir Capacidade Programada. La selecione todas as unidades e continuar, depois selecione a unidade a ser configurada e adicione procedimentos para a mesma com um lime para execução, salve. Após vá até Configuração > Configuração de solicitação – Concluir e depois em Configuração de solicitação – Ativar. Após estes passos é só criar as vagas para o profissional.
